⚙️ Cấu hình Hanet Developer

📊 Trạng thái cấu hình hiện tại

Cấu hình Trạng thái Giá trị Hành động

📋 Hướng dẫn cấu hình

  1. Truy cập Hanet Developer Portal: https://partner.hanet.ai/
  2. Đăng nhập với tài khoản Hanet của bạn
  3. Tạo ứng dụng mới (Create New App)
  4. Lấy thông tin:
    • Client ID
    • Client Secret
    • Access Token
    • Refresh Token
  5. Nhập thông tin vào form trên và nhấn "💾 Lưu cấu hình"
  6. Cấu hình Webhook URL trong Hanet Portal: http://192.168.11.114:1888/hanet-webhook
  7. Test kết nối bằng nút "🔍 Test kết nối"
💡 Lưu ý: Cấu hình sẽ được lưu tự động vào file .env và áp dụng ngay lập tức. Không cần restart server.
